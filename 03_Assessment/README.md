# 03_Assessment

This folder contains grading, rubric, checklist, reflection, and assessment documents.

## Belongs Here

- Employability rubrics.
- Lab rubrics.
- Summative assessment tools.
- Practical exams.
- Line checks.
- Reflection forms.
- Portfolio tools.
- Skill checklists.
- Assessment snapshots.

## Does Not Belong Here

- Full curriculum guides unless assessment is the main purpose.
- Bistro menus.
- Recipes.
- Safety SOPs.
- AI prompt libraries.

## Relationship to the Core Manual

Assessment must use the current grading structure when weights are included: 20% Employability, 35% Labs, and 45% Summative & Bistro.

## Caution

Do not use older grading structures as current policy. Older references such as 10/20/30/40, 25/35/40, or 30/30/40 are historical unless reapproved.
