# Assessment System

## Purpose
Provide a consistent system for evaluating culinary knowledge, technical skill, employability, safety, reflection, and growth.

## Assessment Philosophy
Assessment should reflect how culinary learning actually happens: repeated practice, visible habits, practical performance, service pressure, cleanup, and reflection. Written tests have a place, especially for safety and credentials, but the core of the program is performance.

Assessment should gather appropriate evidence across the three levels defined in the [Professional Thinking Framework](../00_Core/Professional_Thinking_Framework.md): knowledge, skill, and professional thinking. Employability Skills are the observable expression of professional habits such as preparation, organization, communication, accountability, stewardship, adaptability, and reflection. They are not a separate personality or attitude measure.

## Current Source-Based Model
The source history emphasizes behavior-based and employability-centered grading with categories such as:
- Uniform and professionalism.
- Teamwork.
- Execution and efficiency.
- Kitchen cleanliness.
- Preparedness and punctuality.
- Initiative and support.

It also emphasizes skill checklists, practical exams, live Bistro grading, and self-reflection.

## Recommended Grade Evidence
| Evidence Type | Use |
|---|---|
| Daily employability score | Readiness, work habits, communication, and professional behavior |
| Practical skill check | Individual competence |
| Lab rubric | Process, product, safety, and cleanup |
| Bistro live grading | Role performance during service |
| Self-reflection/self-grade | Ownership and improvement planning |
| Credential checks | Food Handler, CKC, or other credential progress |
| Written checks | Vocabulary, kitchen math, safety, recipe analysis |

This table does not set gradebook percentages.

## Bistro Live Grading
The source history records a need to avoid grading for an hour after Friday service. Version 2.1 recommends a lightweight live system:
- Pre-service readiness.
- Mid-service performance.
- Close/reset.
- Reflection submitted.
- Brief note only when needed.

Use a laminated grid, tablet form, or app if available. Station leaders and auditors may provide evidence, but instructor/co-teacher make final scores.

## Simple 4-Level Practical Rubric
| Level | Meaning |
|---|---|
| 4 | Independent, safe, clean, accurate, efficient, helps the team |
| 3 | Competent with minor reminders |
| 2 | Partially competent; needs frequent support or misses key steps |
| 1 | Unsafe, unprepared, incomplete, or unable to demonstrate |

## Current Grade Structure

For 2026-2027, the instructor-confirmed quarter grade structure is:

- 45% Summative Assessments: Projects, Tests, and Bistro.
- 35% Formative Assessments and Labs.
- 20% Employability Skills.

Employability Skills are maintained as a 100-point weekly score using named deductions from the current orientation materials; the exact deduction table must be transferred and verified before it is reproduced here.
