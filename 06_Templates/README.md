# 06_Templates

This folder contains reusable forms, blank templates, checklists, and document starters.

## Belongs Here

- Blank lesson plan templates.
- Bistro meeting templates.
- Station checklist templates.
- Decision note templates.
- Reusable forms.
- Generic document starters.

## Does Not Belong Here

- Completed weekly plans.
- Current policy.
- Final assessment records.
- Historical drafts.

## Relationship to the Core Manual

Templates should make Core Manual expectations easier to use. They should not create new policy.

## Caution

When a template becomes an official assessment, SOP, or curriculum document, move or copy the completed version into the correct operating folder.
