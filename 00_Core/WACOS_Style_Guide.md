# WACOS Style Guide

West Ashley Culinary Operating System

Current as of July 4, 2026

## Purpose

This guide keeps West Ashley Culinary documents, lessons, public materials, AI tools, and communications consistent. It supports the Core Manual; it does not create new program policy.

Use this guide when writing lesson plans, SOPs, Bistro documents, student-facing materials, parent/admin communication, AI prompts, rubrics, public website copy, and program updates.

## Program Voice

West Ashley Culinary writes in a voice that is practical, professional, direct, and grounded in real work.

The voice should sound like a serious culinary and hospitality program inside a high school: clear enough for students, credible enough for families and administrators, and useful enough for daily kitchen operations.

The program voice should not sound academic, promotional, trendy, vague, or overly polished.

## Tone Standards

Use:

- Clear expectations.
- Short paragraphs.
- Plain language.
- Concrete kitchen examples.
- Professional warmth.
- Direct action verbs.
- Calm authority.

Avoid:

- Educational jargon that does not help the work.
- Inflated claims.
- Restaurant-school assumptions.
- Shame-based language.
- Vague inspiration without action.
- Treating students like employees or college culinary students.

## Core Identity Language

Use these terms consistently:

- West Ashley Culinary.
- West Ashley High School Culinary Arts Program.
- Classroom to Table.
- High School, High Standards.
- Kitchen Mode.
- The Bistro or Bistro.
- Employability.
- Fundamentals under pressure.

## Preferred Terminology

| Use | Avoid | Notes |
|---|---|---|
| West Ashley Culinary | WA culinary, culinary class, the cooking class | Use full name in formal documents; shorter references are fine after first use |
| Classroom to Table | farm-to-table unless literally meant | Program identity language |
| High School, High Standards | college-level, professional chef school | Maintains rigor while honoring student reality |
| Bistro | restaurant project, cafe activity, fundraiser | The Bistro is the central instructional model |
| Employability | participation, attitude points | Employability includes observable professional habits |
| Labs | activities, cooking days | Use when work is assessed through practical kitchen performance |
| Summative | final project/test only | Summative includes practical demonstrations and major performance evidence |
| Fundamentals under pressure | advanced class, harder recipes | Preferred Culinary 2 framing |
| Station readiness | setup | More precise for kitchen work |
| Mise en place | getting stuff ready | Use with brief explanation for students when needed |
| Clean-as-you-go | clean up later | Cleanliness is part of performance |
| Reset | cleanup | Reset means the room is ready for the next group |
| Guest | customer | Guest reinforces hospitality |
| Instructor review | AI-approved, automatic, final | AI does not approve program decisions |
| Historical ProStart reference | current ProStart pathway | ProStart Certificate of Achievement is not current program direction |

## Terms to Avoid or Use Carefully

Avoid calling Culinary 2 "advanced" without explanation. Use "full-year depth and operations course" or "fundamentals under pressure."

Avoid calling the Bistro a "fundraiser." It may support program needs, but its instructional role comes first.

Avoid using "participation" as a substitute for Employability. Employability should refer to observable readiness, professionalism, teamwork, communication, initiative, and reset habits.

Avoid "soft skills" when "professional habits," "communication," "hospitality," or "Employability" is more precise.

Avoid "AI says" as a source of authority. Use "drafted with AI and reviewed by the instructor" when needed.

## Naming Conventions

Use clear file names with underscores:

- `Culinary_1_Pacing_Guide.md`
- `Bistro_Prep_Checklist.md`
- `Friday_Service_Reflection.md`
- `Knife_Safety_SOP.md`

Use title case for document titles:

- Bistro Prep Checklist
- Culinary 2 Practical Assessment
- Kitchen Mode Routine

Use version numbers for governing documents:

- Version 1.0
- Version 1.1

Use dates for time-bound documents:

- Week of 2026-08-24
- Revised 2026-07-04

## Student-Facing Language

Student-facing language should be direct, concrete, and usable during class.

Use:

"Enter Kitchen Mode: uniform checked, hands washed, notebook out, station assigned."

"Your station is not finished until it is reset for the next class."

"Employability is how you show you can be trusted in a kitchen."

Avoid:

"Students will demonstrate comprehensive mastery of professional dispositions across authentic learning contexts."

"Have a good attitude."

"Clean up."

## Parent/Admin-Facing Language

Parent/admin-facing language should explain the seriousness of the program without overselling it.

Use:

"West Ashley Culinary uses the Bistro as a real instructional service model where students practice safety, sanitation, cooking, communication, hospitality, and responsibility."

"The current grading model is 30% Employability, 30% Labs, and 40% Summative evidence."

Avoid:

"Students run a restaurant just like industry professionals."

"This program guarantees career readiness."

"The Bistro is our fundraiser."

## AI-Generated Writing Expectations

AI-generated writing must be reviewed before use.

AI writing should:

- Follow the Core Manual.
- Use current WACOS terminology.
- Avoid invented policy.
- Flag unclear decisions.
- Distinguish current practice from historical material.
- Use practical language.
- Preserve instructor authority.

AI writing should not:

- Make final safety, allergen, grading, discipline, legal, or compliance decisions.
- Present archived material as current policy.
- Revive obsolete ProStart-centered language.
- Use generic culinary-school assumptions.
- Add unnecessary jargon.

## Strong vs. Weak Language

| Strong | Weak |
|---|---|
| "The Bistro is the central instructional model." | "The Bistro is a fun project students do." |
| "Students are expected to be ready, clean, organized, safe, useful, and coachable." | "Students should try to do their best." |
| "Culinary 2 develops fundamentals under pressure." | "Culinary 2 is the advanced class." |
| "Employability measures observable professional habits." | "Participation is based on attitude." |
| "AI may draft a prep list for instructor review." | "AI can make the prep plan." |
| "This item needs instructor confirmation." | "We can assume this is the policy." |

## Final Style Screen

Before approving a document, ask:

- Does it sound like West Ashley Culinary?
- Does it support Classroom to Table and High School, High Standards?
- Does it treat the Bistro as central?
- Does it use current grading and credential language?
- Does it avoid obsolete ProStart-centered language?
- Does it make unclear items visible instead of inventing policy?
- Would this help someone do the work tomorrow?
