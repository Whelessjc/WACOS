# Glossary

## Standard Terms
| Term | Use |
|---|---|
| West Ashley Culinary | Preferred program name |
| WACOS | West Ashley Culinary Operating System |
| Kitchen + Classroom | Public-facing identity phrase |
| Culinary 1 | One-semester foundation and exploration course |
| Culinary 2 | Full-year depth and operations course |
| Hospitality Work-Based Learning | Application stage after or alongside advanced pathway work |
| Bistro | Student-led restaurant/service experience |
| Food Handler | Culinary 1 credential target unless updated |
| CKC / Certified Kitchen Cook | Culinary 2 credential alignment target unless updated |
| Fundamentals under pressure | Culinary 2 instructional philosophy |
| Pre-Bistro fundamentals | Minimum operational skills students need before public service |
| Live grading | During-service or same-day observational scoring |
| Source-bound AI | AI use constrained to provided or verified program sources |

## Terms To Avoid Or Qualify
| Term | Guidance |
|---|---|
| Advanced | Avoid as the main description for Culinary 2 unless explaining that the course is deeper, not simply fancier. |
| Culinary 3 | Use only as informal shorthand; official language should be Hospitality Work-Based Learning until confirmed. |
| ServSafe Manager | Retired from the current Culinary 2 credential pathway; use CKC instead. |
