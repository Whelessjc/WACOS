# 00_Core

This folder contains the governing documents for WACOS.

## Belongs Here

- Core Manual.
- Playbook.
- Style Guide.
- Alignment Checklist.
- Traceability.
- AI Reference.
- Custom Instructions Draft.
- Decision registers.
- Source manifests.
- Program philosophy and standards.
- Completion reports that explain governing-document creation.

## Does Not Belong Here

- Daily lesson plans.
- Recipes.
- Menus.
- Equipment-specific SOPs.
- Historical source packets.
- Duplicate drafts.

## Relationship to the Core Manual

The Core Manual governs the rest of the repository. When another document conflicts with this folder, use the Core Manual and current instructor-confirmed decisions first.

## Caution

Do not change policy in this folder casually. Major edits should update traceability, decision records, and the changelog.
