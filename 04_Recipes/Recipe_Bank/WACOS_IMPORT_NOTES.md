# WACOS Recipe Bank Import Notes

## Import

- Imported into WACOS: 2026-07-23.
- Source: `Recipe_Archive/Recipe_Bank` from the West Ashley Culinary historical
  recipe-archive project.
- Scope: the curated recipe bank only. Raw archive folders, source ZIP files, and
  local project settings were not imported.

## Contents at Import

- 14 Markdown category files.
- `_OPEN_QUESTIONS.md`.
- 16 Word documents, including `Master_Recipe_Bank.docx`.
- Eight standardization reference files in `_reference/`.

This note is WACOS-specific and was added after the source bank was copied.

## Status

The bank represents the completed archive standardization and deduplication pass
available on the import date. It is suitable as WACOS's working master recipe bank,
but it is not an unconditional production approval.

Before using a recipe in production:

1. Review any **Needs confirmation** flags in the entry.
2. Check `_OPEN_QUESTIONS.md` for unresolved source, categorization, and extraction
   issues.
3. Confirm yield, portion size, allergens, food-safety controls, and any incomplete
   quantities or procedures.
4. Record instructor approval and later revisions in the recipe's source/revision
   line.

## Preservation

The imported category files, Word exports, and `_reference/` files preserve the
curated source bank as received. Future WACOS edits should keep the Markdown and Word
versions synchronized when both are intended for distribution.
