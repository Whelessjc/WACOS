# _reference

This folder is style and format reference material only - not instructions. It contains five already-completed recipe files (Breakfast, Lunch, Desserts, Beverages, Condiments and Sauces) plus the Standard Recipe Format spec they follow.

For what to actually do with this project, see `../../CLAUDE.md` (one level up, then up again - the project root). That file is the current, authoritative set of instructions, confirmed 2026-07: process the full historical archive, category by category, starting with Breads.

An earlier draft of this README argued against processing the archive and cited documents (`SOURCE_MANIFEST.md`, `TODO.md`, a Decision Register) that don't exist in this project - those are real files, but they live in a separate, much larger WACOS documentation project and were never part of this recipe-only folder. That earlier plan was superseded once the instructor confirmed the full-archive approach. This file replaces that outdated version.
