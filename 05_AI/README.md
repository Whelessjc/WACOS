# 05_AI

This folder contains AI-related WACOS documents.

## Belongs Here

- AI workflows.
- Prompt libraries.
- AI tutor designs.
- Custom GPT instructions.
- Automation concepts.
- AI review guidance.

## Does Not Belong Here

- Student data.
- Unreviewed private transcripts intended for student-facing use.
- Final safety policy.
- Final grading decisions.
- District policy.

## Relationship to the Core Manual

AI supports the program but does not govern it. AI must follow the Core Manual and the source-of-truth hierarchy.

## Caution

AI-generated recipes, assessments, safety instructions, parent/admin communications, and student-facing tools require instructor review.
