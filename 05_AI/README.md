# 05_AI

This folder contains AI-related WACOS documents.

## Belongs Here

- AI workflows.
- Prompt libraries.
- AI tutor designs.
- Custom GPT instructions.
- Automation concepts.
- AI review guidance.

## Does Not Belong Here

- Student data.
- Unreviewed private transcripts intended for student-facing use.
- Final safety policy.
- Final grading decisions.
- District policy.

## Relationship to the Core Manual

AI supports the program but does not govern it. AI must follow the Core Manual and the source-of-truth hierarchy.

## ChatGPT Project Knowledge

Upload [`WACOS_ChatGPT_Quick_Reference.md`](WACOS_ChatGPT_Quick_Reference.md) to the WACOS ChatGPT Project for fast answers to common program questions. Use the connected GitHub repository for detailed procedures, citations, recent changes, and questions not covered by the quick reference.

When a permanent decision changes a high-frequency fact in the quick reference, update the governing document and the quick reference in the same published change.

## Caution

AI-generated recipes, assessments, safety instructions, parent/admin communications, and student-facing tools require instructor review.
