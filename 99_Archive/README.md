# 99_Archive

This folder contains historical, superseded, retired, duplicate, unclear-current-status, and raw source materials.

## Belongs Here

- Research archives.
- Source packets.
- Historical WACOS versions.
- Release reports.
- Migration reports.
- Generated export copies.
- Duplicate source copies.
- Old ProStart-centered documents.
- Unclear-current-status materials.

## Does Not Belong Here

- Current operating documents.
- Current student-facing documents.
- Current safety SOPs.
- Current grading tools.

## Relationship to the Core Manual

Archive materials may explain context and history. They do not override the Core Manual or current instructor-confirmed decisions.

## Caution

Do not treat archived material as current policy unless it is explicitly reviewed and reapproved. Be especially careful with older ProStart references, older grading structures, and generated drafts.
