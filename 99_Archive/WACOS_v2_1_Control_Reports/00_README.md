# West Ashley Culinary Operating System (WACOS) Version 2.1

## Purpose
WACOS is the operating knowledge base for the West Ashley High School Culinary Program. It supports teacher onboarding, AI tutors, custom GPT knowledge, curriculum planning, Bistro operations, student leadership, lesson development, assessments, and long-term program documentation.

## What Changed in 2.1
Version 2.1 is a quality-assurance release. It keeps the Version 2.0 reconstruction, but separates operating documentation from raw evidence, centralizes unresolved decisions, standardizes terminology, and adds missing control documents.

## Evidence and Archive
Raw source packets, topic packets, and extraction reports are intentionally kept out of this operating repository. They belong in `WestAshleyCulinary_Research_Archive_v2_1`.

## Use Rules
- Treat core documents as the current working manual.
- Treat [DECISION_REGISTER.md](DECISION_REGISTER.md) as the single source for unresolved conflicts and instructor-review decisions.
- Use [SOURCE_MANIFEST.md](SOURCE_MANIFEST.md) for traceability and the research archive for deeper evidence.
- Do not upload raw private transcripts into student-facing AI systems without review and cleaning.
