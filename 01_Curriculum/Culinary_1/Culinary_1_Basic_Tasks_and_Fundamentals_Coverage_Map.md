# Culinary 1 - Basic Tasks & Fundamentals Coverage Map

## What the Scan Found

Culinary 1's 11-unit package is strong at the habit and concept level. Every unit checklist already tracks things like "I can use and clean a thermometer," "I can explain cleaning vs. sanitizing," and "I know how my station affects the guest or next teammate." Personal hygiene, allergen awareness, and cross-contamination are well represented across the package (100+ file hits each for handwashing, allergens, sanitizer).

What's consistently missing is the **procedural layer underneath those habits** - the specific, teachable micro-skill a student needs to actually perform the habit correctly, not just know it matters. A search across every unit turned up zero hits for: hand sink setup, three-compartment sink procedure detail, sanitizer ppm/test strips, thermometer calibration method, ice bath cooling procedure, thawing methods, FIFO by name, date-marking mechanics, receiving procedure, smallwares care, and the acronym-level Bistro lingo ("86," FOH/BOH).

That matches exactly what you described: these are the tasks most likely to be picked up by osmosis rather than taught and checked off deliberately. The fix isn't a new curriculum - it's making sure each one has a named moment where it's explicitly taught, practiced, and signed off, inside the units that already exist.

## Recommended Mechanism: A Basic Tasks Sign-Off Card

Rather than adding a parallel curriculum, give each student one running card (paper or digital) that lists every procedural item from the map above in plain language, grouped by the unit where it's taught. The teacher (or a trained student leader) initials it the first time a student demonstrates the task correctly - not when it's merely discussed.

This does three things the current unit checklists don't:

1. Makes coverage auditable. You can look at a stack of cards partway through the semester and see exactly which tasks are lagging class-wide, instead of assuming they've been absorbed.
2. Separates "discussed" from "demonstrated." A student can nod along to a sanitizer-ppm mini-lesson without ever actually mixing sanitizer and checking it with a test strip themselves - the card only gets initialed for the second one.
3. Gives Bistro readiness a real gate. Before Unit 4 (Customer Service and Bistro Soft Opening), the card should show every Unit 1-3 item initialed. That's a concrete readiness check, not a calendar date.

I can build this card as an actual student-facing artifact (one page, organized by unit, checkbox format) if that's useful alongside the flex-mod calendars.
