## Entering the Kitchen
- [ ] Enter calmly and put belongings away.
- [ ] Check agenda and readiness.
- [ ] Join the huddle.
- [ ] Wash hands when directed.
- [ ] Begin only when released.

## Station Assignment
- [ ] I know my role.
- [ ] I know my tools.
- [ ] I know my setup.
- [ ] I know what must be ready first.
- [ ] I know my ticket/task flow.
- [ ] I know my reset job.
- [ ] I know who to ask when unsure.

## Prep Responsibility
- [ ] Prep list checked.
- [ ] Ingredients gathered as directed.
- [ ] Tools organized.
- [ ] Timing needs named.
- [ ] Items labeled or stored correctly.
- [ ] Station kept clean enough to work.

## Communication
- [ ] Use "heard," "working," and "order up" when needed.
- [ ] Call back key ticket information.
- [ ] Ask for help before the station falls behind.
- [ ] Speak respectfully.

## Ticket Flow
- [ ] Ticket is readable.
- [ ] Ticket goes to the correct station.
- [ ] Ticket stays visible.
- [ ] Expo checks before handoff.
- [ ] Ticket/order is completed and reset.

## Service Behavior
- [ ] Stay in role unless reassigned.
- [ ] Move calmly and safely.
- [ ] Notice guest space.
- [ ] Support teammates.
- [ ] Reset small things during service.
- [ ] Debrief systems, not people.

## Sanitation
- [ ] Hands washed when needed.
- [ ] Counters cleaned and sanitized.
- [ ] Clean and dirty items separated.
- [ ] Dish flow active.
- [ ] Spills, towels, and floors controlled.

## Reset
- [ ] Tickets/tasks accounted for.
- [ ] Station reset.
- [ ] Service area reset.
- [ ] Dishes routed correctly.
- [ ] Tools and towels returned correctly.
- [ ] Trash and floor checked.
- [ ] Station ready for the next group.

## Exit Routine
- [ ] Station inspected.
- [ ] Reflection or debrief completed.
- [ ] Belongings collected.
- [ ] Leave calmly when dismissed.

## Before You Turn This In
- [ ] My name, date, and class period are filled in.
- [ ] I completed the assigned checks or reflection questions.
- [ ] I understand my cleanup/reset responsibility for this activity.
