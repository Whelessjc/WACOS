# Create Google Slides Instructions

Use the files in `FINAL-DELIVERY/Google-Slides-Ready` as slide source copy.

## Copy/Paste Process
1. Open Google Slides.
2. Create a new deck for the unit.
3. Copy each slide title into a slide title box.
4. Copy the `On-Slide Text` bullets into the body area.
5. Use the `Suggested Visual` note to choose a photo, icon, or simple diagram.
6. Copy `Teacher Notes` into speaker notes.
7. Use `Activity Prompt` as a quick check, discussion, or movement cue.

## Start With
- Unit 01 first.
- Review slide spacing and visual style.
- Apply the same style to Units 02 through 11.
