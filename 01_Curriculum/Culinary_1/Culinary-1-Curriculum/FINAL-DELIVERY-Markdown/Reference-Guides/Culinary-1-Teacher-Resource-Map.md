# Culinary 1 Teacher Resource Map

## Purpose
This map helps the teacher use the final delivery folder for printing, Google Drive upload, Google Docs conversion, Google Slides creation, and classroom use today.

## Final Delivery Folder Map
- `Google-Docs-Ready`: Markdown files and DOCX files for Google Docs or Word.
- `Printer-Ready-PDFs`: PDF versions of the main printable packets.
- `Google-Slides-Ready`: slide source Markdown and PPTX files for Google Slides.
- `Teacher-Use`: complete teacher planning packet.
- `Student-Use`: complete student packet and checklist bundle.
- `Lab-Sheets`: complete lab/service sheet bundle.
- `Reference-Guides`: student quick reference and teacher resource map.
- `Reports`: upload, printing, conversion, and final delivery reports.

## What To Print First
1. Student quick reference guide.
2. Unit 01 student packet pages if starting immediately.
3. Unit 01 pancake lab sheet.
4. Checklist bundle pages as weekly handouts.
5. Lab sheet bundle pages as needed by unit.

## What Still Needs Teacher Review
- Exact recipes, formulas, yields, and ingredient quantities.
- School-specific uniform policy.
- Local dish machine and three-compartment sink procedures.
- Local fryer access and oil disposal policy.
- Approved protein products and safe-temperature language.
- Culinary 2 readiness or recommendation process.

## Recommended Use Today
Start with Unit 01. Print the student quick reference, Unit 01 checklist, and Unit 01 pancake lab sheet. Upload the full student packet and teacher packet to Google Drive for review and sharing.
