# Checklist: Protein as Application

## Readiness Checklist
- [ ] I know the protein safety rules.
- [ ] I know which board/tools to use.
- [ ] I know the temperature goal if required.
- [ ] I know when to call the teacher for a check.

## Daily Habits
- [ ] Wash hands after touching raw protein.
- [ ] Keep raw and clean items separate.
- [ ] Clean thermometer correctly.
- [ ] Communicate around hot pans.
- [ ] Sanitize after protein work.
- [ ] Keep raw tools away from finished food.
- [ ] Communicate when protein is cooking, resting, checked, or ready.

## Skill Checks
- [ ] Egg cookery
- [ ] Ground turkey cooking
- [ ] Chicken setup
- [ ] Searing
- [ ] Temperature checking
- [ ] Resting
- [ ] Slicing
- [ ] Raw/clean reset
- [ ] Holding or service timing awareness

## Cleaning and Reset Checks
- [ ] Raw protein area sanitized.
- [ ] Thermometer cleaned.
- [ ] Boards and knives washed.
- [ ] Counters sanitized.
- [ ] Floor checked.

## Student Self-Check
- [ ] I prevented cross-contamination.
- [ ] I checked doneness correctly.
- [ ] I can name one earlier skill I used.
