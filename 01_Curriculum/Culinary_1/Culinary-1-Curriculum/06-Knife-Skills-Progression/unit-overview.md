# Unit 06: Knife Skills Progression

## Weeks Covered
Weeks 6 to 9

## Focus
Build confidence through control and repetition.

## Purpose
Students build safe knife control through repeated practice connected to real prep. Skills include knife safety, grip, claw, board setup, rough chop, dice, slice, mince, julienne, batonnet, speed with control, prep list execution, stir fry, sauteed vegetables, soup base, roasted vegetables, potato batonnet, fries, board/station stewardship, and a Week 9 deep clean reset.

## Learning Targets
- I can set up a safe cutting station.
- I can use a stable grip and claw.
- I can complete basic cuts with control.
- I can improve speed without losing safety.
- I can use knife skills to complete a prep list.
- I can help deep clean and reset the kitchen.

## Core Skills
- Knife carrying and storage
- Board setup
- Grip and claw
- Rough chop, slice, dice, mince
- Julienne and batonnet
- Knife-to-prep-list execution
- Deep clean reset
- Board and knife station stewardship

## Vocabulary
| Term | Student-Friendly Meaning |
|---|---|
| Grip | How the knife hand holds the knife. |
| Claw | Guiding hand shape that protects fingers. |
| Dice | Cut into small cube-like pieces. |
| Mince | Cut into very small pieces. |
| Julienne | Thin matchstick cut. |
| Batonnet | Thicker stick cut. |

## Lab Applications
- Stir fry
- Sauteed vegetables
- Soup base
- Roasted vegetables
- Potato batonnet
- Fries
- Week 9 deep clean reset

## Bistro Connection
Knife skills make prep faster, safer, more consistent, and more useful for service. Bistro prep also depends on clean boards, sorted product, labeled storage, and safe knife cleanup.

## Assessment Ideas
- Knife station setup check
- Grip and claw observation
- Cut consistency check
- Prep list completion
- Deep clean participation

## Teacher Notes
Do not reward speed without control. Students need repeated, calm practice and immediate correction for unsafe habits.

## Resources Needed
- Equipment: knives, boards, damp towels, bowls, bench scrapers, sheet pans, pans, sanitizer
- Ingredients: practice vegetables, potatoes, soup base vegetables, stir fry vegetables
- Materials: prep lists and cut examples if available
