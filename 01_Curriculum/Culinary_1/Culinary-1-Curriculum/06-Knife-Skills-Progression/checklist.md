# Checklist: Knife Skills Progression

## Readiness Checklist
- [ ] I have permission to use a knife.
- [ ] My board is stable.
- [ ] My station is clear.
- [ ] I know the cut I am practicing.

## Daily Habits
- [ ] Carry knife safely.
- [ ] Keep eyes on the cut.
- [ ] Use grip and claw.
- [ ] Slow down when control slips.
- [ ] Keep board organized.
- [ ] Separate usable product, scraps, and trash.

## Skill Checks
- [ ] Rough chop
- [ ] Slice
- [ ] Dice
- [ ] Mince
- [ ] Julienne
- [ ] Batonnet
- [ ] Prep list execution
- [ ] Product sorted and stored correctly

## Cleaning and Reset Checks
- [ ] Knife cleaned and stored.
- [ ] Board washed.
- [ ] Counter sanitized.
- [ ] Food handled as directed.
- [ ] Floor checked.
- [ ] Usable prep labeled or stored as directed.

## Student Self-Check
- [ ] I worked safely.
- [ ] My cuts improved with practice.
- [ ] I know which cut needs more repetition.
