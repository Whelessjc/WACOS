# Checklist: Recipe Reading and Mise en Place

## Readiness Checklist
- [ ] I completed the entry routine.
- [ ] I washed hands.
- [ ] I have the recipe visible.
- [ ] I know my team role.

## Daily Habits
- [ ] Read before touching ingredients.
- [ ] Ask if a step is unclear.
- [ ] Keep tools organized.
- [ ] Clean as I work.
- [ ] Take only what my group needs and return shared items correctly.

## Skill Checks
- [ ] I can identify ingredients.
- [ ] I can identify equipment.
- [ ] I can identify yield.
- [ ] I can sequence steps.
- [ ] I can make a simple prep list.
- [ ] I can set up mise en place.

## Cleaning and Reset Checks
- [ ] Ingredients stored or returned.
- [ ] Dishes handled correctly.
- [ ] Counters sanitized.
- [ ] Tools returned.
- [ ] Floor checked.
- [ ] Shared ingredients handled with care.

## Student Self-Check
- [ ] I read before acting.
- [ ] I helped organize the station.
- [ ] I know what I would prep earlier next time.
- [ ] I left the station ready for another group.
