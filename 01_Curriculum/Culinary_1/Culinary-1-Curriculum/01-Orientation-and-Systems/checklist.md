# Checklist: Orientation and Systems

## Readiness Checklist
- [ ] I know the daily entry routine.
- [ ] I understand the uniform/readiness expectations.
- [ ] I know where belongings go.
- [ ] I know when to wash my hands.
- [ ] I know how to join the agenda check and huddle.

## Teacher First-Week Prep
- [ ] Course purpose is ready: real cooking, real skills.
- [ ] Uniform expectations are ready to explain.
- [ ] Kitchen walk-through route is planned.
- [ ] Measuring stations are ready.
- [ ] Pancake ingredients and equipment are ready.
- [ ] Cleanup/reset routine is ready to model before Friday.

## Daily Habits
- [ ] Enter calmly.
- [ ] Check the agenda.
- [ ] Listen during huddle.
- [ ] Use kitchen communication.
- [ ] Ask before using equipment.
- [ ] Walk and keep pathways clear.
- [ ] Use "behind," "hot," "sharp," "corner," and "heard" when needed.

## Skill Checks
- [ ] I can find major kitchen areas.
- [ ] I can measure dry ingredients.
- [ ] I can measure liquid ingredients.
- [ ] I can set up a basic station.
- [ ] I can read directions before starting.
- [ ] I can name one heat cue from the pancake lab.

## Cleaning and Reset Checks
- [ ] Dishes are handled correctly.
- [ ] Counters are wiped and sanitized.
- [ ] Tools are returned.
- [ ] Towels are handled correctly.
- [ ] Trash and floor are checked.
- [ ] Station is ready for the next group.

## Student Self-Check
- [ ] I helped my team.
- [ ] I worked safely.
- [ ] I did not rush past directions.
- [ ] I cleaned before leaving.
- [ ] I know one routine I need to practice again.

## End-of-Week Teacher Reflection
- [ ] Which routines are students already following?
- [ ] Which routines need reteaching next week?
- [ ] Did measuring stations prepare students for the pancake lab?
- [ ] Was Friday's pancake lab manageable?
- [ ] What cleanup issue needs to be addressed first?
