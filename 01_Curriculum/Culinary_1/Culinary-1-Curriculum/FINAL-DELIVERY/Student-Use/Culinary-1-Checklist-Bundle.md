# Unit 01 Checklist: Orientation and Systems

Name: ________________________________    Date: ________________    Class Period: ________

## Unit Focus
Learn the room before cooking.

## Entering the Kitchen
- [ ] Enter calmly.
- [ ] Put belongings away.
- [ ] Check the agenda.
- [ ] Go to the assigned starting area.
- [ ] Listen during huddle and wait to begin.

## Uniform
- [ ] Follow school uniform expectations.
- [ ] Hair restrained as directed.
- [ ] Safe shoes.
- [ ] Loose items and jewelry follow class policy.
- [ ] Ready to work.

## Handwashing
- [ ] Wash before food work.
- [ ] Wash after touching face, hair, phone, trash, floor, or dirty dishes.
- [ ] Wash after restroom use, coughing, sneezing, or blowing nose.
- [ ] Wash any time the teacher calls for a reset.

## Station Setup
- [ ] Directions read.
- [ ] Team roles assigned.
- [ ] Tools gathered.
- [ ] Ingredients gathered or received.
- [ ] Towel, trash, and dish plan clear.
- [ ] Counter clean and usable.

## Measuring
- [ ] Correct measuring tool selected.
- [ ] Dry ingredients measured carefully.
- [ ] Liquid measurements read at eye level.
- [ ] Tools returned when finished.

## Lab Behavior
- [ ] Walk in the kitchen.
- [ ] Use "behind," "hot," "sharp," "corner," and "heard."
- [ ] Ask before using equipment.
- [ ] Keep towels away from heat.
- [ ] Work with your team and clean as you go.

## Cleanup and Reset
- [ ] Heat source off as directed.
- [ ] Dishes handled correctly.
- [ ] Counters wiped and sanitized.
- [ ] Tools returned.
- [ ] Towels handled correctly.
- [ ] Trash and floor checked.
- [ ] Station ready for the next group.

## Exit Routine
- [ ] Station inspected.
- [ ] Reflection completed if assigned.
- [ ] Belongings collected.
- [ ] Leave calmly when dismissed.

## Student Self-Check
- [ ] I entered class ready to work.
- [ ] I followed safety expectations.
- [ ] I supported my station or team.
- [ ] I helped leave the kitchen ready for the next group.

- [ ] Reader tracks the next step.
- [ ] Steward starts cleanup early.

## Lab Behavior

# Unit 04 Checklist: Customer Service and Bistro Soft Opening

Name: ________________________________    Date: ________________    Class Period: ________

## Unit Focus
Understand the full flow of service.

## Entering the Kitchen
- [ ] Enter calmly and put belongings away.
- [ ] Check the agenda and uniform/readiness.
- [ ] Join the huddle.
- [ ] Wash hands when directed.
- [ ] Begin only when released.

## Station Assignment
- [ ] I know my role.
- [ ] I know my tools.
- [ ] I know my setup.
- [ ] I know my ticket/task flow.
- [ ] I know my reset job.
- [ ] I know who to ask when unsure.

## Life of an Order
- [ ] Guest arrives and is greeted.
- [ ] Order is taken and ticket moves.
- [ ] Food/service item is prepared.
- [ ] Expo checks before handoff.
- [ ] Food is run or handed off.
- [ ] Check back, clear, dish, reset, and farewell.

## Communication
- [ ] Use "behind," "hot," "corner," and "heard."
- [ ] Call back key ticket information.
- [ ] Ask for help before the station falls behind.
- [ ] Speak respectfully.

## Guest Service
- [ ] Greet respectfully.
- [ ] Use a calm voice.
- [ ] Listen before responding.
- [ ] Notice guest space.
- [ ] Follow through.

## Ticket Flow
- [ ] Ticket is readable.
- [ ] Ticket goes to the correct station.
- [ ] Ticket stays visible.
- [ ] Ticket/order is completed and reset.

## Mock Service / Soft Opening
- [ ] I stay in role unless reassigned.
- [ ] I communicate before the station falls behind.
- [ ] I help dish and reset before being asked.
- [ ] I debrief the system, not the person.

## Sanitation
- [ ] Hands washed when needed.
- [ ] Stations kept clean.
- [ ] Dishes moving.
- [ ] Spills, towels, and floors controlled.

## Reset
- [ ] Guest area reset.
- [ ] Service area reset.
- [ ] Tickets/tasks accounted for.
- [ ] Dishes routed correctly.
- [ ] Counters cleaned and sanitized.
- [ ] Tools and towels returned correctly.
- [ ] Trash and floor checked.
- [ ] Station ready for the next group.

## Exit Routine
- [ ] Station inspected.
- [ ] Debrief or reflection completed.
- [ ] Belongings collected.
- [ ] Leave calmly when dismissed.

## Student Self-Check

- [ ] I followed safety expectations.

# Unit 06 Checklist: Knife Skills Progression

Name: ________________________________    Date: ________________    Class Period: ________

## Unit Focus
Build confidence through control and repetition.

## Entering the Kitchen
- [ ] Enter calmly and put belongings away.
- [ ] Check agenda and readiness.
- [ ] Join the huddle.
- [ ] Wait for teacher direction before knives are issued.

## Uniform and Handwashing
- [ ] Uniform/readiness checked.
- [ ] Hair restrained as directed.
- [ ] Hands washed.
- [ ] Towel and sanitation plan clear.

## Board Setup
- [ ] Damp towel under board.
- [ ] Board stable.
- [ ] Knife placed safely.
- [ ] Scrap/waste plan ready.
- [ ] Usable prep container ready.

## Knife Safety
- [ ] Knife down during directions.
- [ ] Carry safely.
- [ ] Keep safe space.
- [ ] Use stop signal.
- [ ] Stop if blade path is unsafe.
- [ ] Ask for help when unsure.

## Grip and Claw
- [ ] Grip is controlled.
- [ ] Claw hand is tucked.
- [ ] Eyes stay on the cut.
- [ ] Product hand moves slowly.
- [ ] Thumb stays behind fingers.

## Controlled Cutting
- [ ] Control before speed.
- [ ] Blade path is safe.
- [ ] Cut size fits the task.
- [ ] Board stays clean enough to work.

## Product Organization
- [ ] Usable prep sorted.
- [ ] Scraps and waste separated.
- [ ] Containers labeled/stored if directed.
- [ ] Prep list checked.

## Communication
- [ ] Use "sharp," "behind," and "knife down" when needed.
- [ ] Ask before the station becomes unsafe.
- [ ] Give teammates safe space.

## Cooking Application
- [ ] Cut size matches the cooking task.
- [ ] Prep supports the recipe or Bistro task.
- [ ] Station timing is considered.
- [ ] Product is labeled/stored if directed.

## Sanitation
- [ ] Counters cleaned and sanitized.
- [ ] Dishes routed correctly.
- [ ] Towels handled correctly.
- [ ] Trash and floor checked.

## Reset
- [ ] Knife cleaned and stored safely.
- [ ] Board washed correctly.
- [ ] Tools returned.
- [ ] Station ready for the next group, prep block, or service block.

## Exit Routine
- [ ] Station inspected.
- [ ] Reflection completed.
- [ ] Belongings collected.
- [ ] Leave calmly when dismissed.

## Student Self-Check
- [ ] I entered class ready to work.
- [ ] I followed safety expectations.
- [ ] I supported my station or team.
- [ ] I helped leave the kitchen ready for the next group.
