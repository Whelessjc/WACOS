# Printer Package Guide

## Recommended Files To Print
- `Culinary-1-Student-Packet.md` or PDF version
- `Culinary-1-Checklist-Bundle.md` or PDF version
- `Culinary-1-Lab-Sheet-Bundle.md` or PDF version
- `Culinary-1-Student-Quick-Reference.md` or PDF version

## Suggested Print Order
1. Student quick reference guide
2. Unit checklists
3. Lab sheets needed first
4. Full student packet if using a binder or packet system

## Single-Sided vs Double-Sided
- Student quick reference: single-sided or laminated if possible.
- Lab sheets: single-sided works best for writing and kitchen use.
- Student packet: double-sided is fine after page breaks are reviewed.
- Checklist bundle: single-sided if used as weekly handouts.

## Review Before Printing
- School-specific uniform policy.
- Recipe quantities and yields.
- Fryer/oil policy.
- Protein temperature language.
- Dish machine and three-compartment sink procedures.
