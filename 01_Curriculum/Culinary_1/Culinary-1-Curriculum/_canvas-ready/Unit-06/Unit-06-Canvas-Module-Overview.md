# Unit-06 Canvas Module Overview: Knife Skills Progression

## Module Title
Unit-06: Knife Skills Progression

## Student-Facing Overview
In this module, you will work on: **Build confidence through control and repetition.** This unit is about real
cooking, real skills, safe habits, teamwork, stewardship, and leaving the kitchen ready for the next group.

## What You Will Learn
- knife safety
- board setup
- grip
- claw
- blade path
- rough chop
- dice
- slice
- mince
- julienne
- batonnet
- deep clean reset

## What You Will Do
- set up a board safely
- practice cuts slowly
- connect cut size to cooking
- support Bistro prep
- complete deep clean reset

## What You Will Turn In
- knife safety check
- practice reflection
- deep clean/reset check

## Materials Needed
- student packet
- knife skills handout
- knife practice lab handout
- slide deck content
