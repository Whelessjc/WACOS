# Checklist: Food Handler and Stewardship

## Readiness Checklist
- [ ] I am in uniform/readiness expectations.
- [ ] I wash hands before food work.
- [ ] I know my stewardship role.
- [ ] I know where dirty dishes, towels, trash, and sanitizer go.

## Daily Habits
- [ ] Keep hands clean.
- [ ] Keep towels in the correct place.
- [ ] Report spills.
- [ ] Keep raw and ready-to-eat items separate.
- [ ] Ask before using chemicals or equipment.

## Skill Checks
- [ ] I can explain cleaning vs. sanitizing.
- [ ] I can use dish flow correctly.
- [ ] I can identify a cross-contamination risk.
- [ ] I can identify an allergen concern.
- [ ] I can use and clean a thermometer.

## Cleaning and Reset Checks
- [ ] Counters cleaned and sanitized.
- [ ] Dishes washed or routed correctly.
- [ ] Laundry/towels handled correctly.
- [ ] Storage checked.
- [ ] Floors checked.
- [ ] Station reset.

## Student Self-Check
- [ ] I did a stewardship job without being pushed.
- [ ] I protected someone else through safe habits.
- [ ] I know one sanitation habit I need to repeat.
