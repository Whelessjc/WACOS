## Mise en Place Checklist
- [ ] Read the full lab sheet before starting.
- [ ] Wash hands.
- [ ] Gather tools and materials.
- [ ] Set up towel, trash, dish, and sanitizer plan.
- [ ] Know your role.
- [ ] Wait for the teacher checkpoint before heat, knives, fryer, or raw protein work.

## Procedure
1. Complete skill inventory using real examples.
2. Create a short prep list for a teacher-selected task.
3. Complete limited ownership task if assigned.
4. Name evidence from lab, service, teamwork, and reset.
5. Complete final stewardship reset.
6. Write readiness reflection.

## Teacher Checkpoints
- [ ] Evidence is specific.
- [ ] Prep list includes tools, timing, safety, and cleanup.
- [ ] Student follows through on assigned role.
- [ ] Final reset is complete.
- [ ] Reflection is honest and practical.

## Cleanup / Reset Checklist
- [ ] Dishes routed correctly.
- [ ] Counters cleaned and sanitized.
- [ ] Tools returned clean and dry.
- [ ] Towels/laundry handled correctly.
- [ ] Trash checked.
- [ ] Floor checked.
- [ ] Food stored or discarded as directed.
- [ ] Station ready for the next group.


## Reflection Questions
1. What skill did you practice?
2. What safety habit mattered most?
3. What correction helped your work?
4. What cleanup/reset job did you complete?
5. What evidence shows you are ready for more independent kitchen work?
