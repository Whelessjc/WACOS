# Unit 03 Printable Checklist

## Entering the Kitchen
- [ ] Enter calmly.
- [ ] Put belongings away.
- [ ] Check the agenda.
- [ ] Check uniform/readiness.
- [ ] Join the huddle and wash hands when directed.
- [ ] Begin only when released.

## Recipe Reading Before Action
- [ ] Read the whole recipe.
- [ ] Find the title and yield.
- [ ] Mark ingredients.
- [ ] Mark equipment.
- [ ] Notice timing and safety notes.
- [ ] Ask questions before touching ingredients.

## Gathering Equipment
- [ ] Measuring tools ready.
- [ ] Mixing/cooking tools ready.
- [ ] Bowls, pans, pots, or rice cooker ready if needed.
- [ ] Dish and trash plan clear.

## Measuring Ingredients
- [ ] Correct tool selected.
- [ ] Dry ingredients measured carefully.
- [ ] Liquid measurements checked.
- [ ] Shared ingredients kept clean.

## Building Mise en Place
- [ ] Team roles assigned.
- [ ] Prep list made.
- [ ] Ingredients and equipment staged.
- [ ] Counter clean and usable.
- [ ] Teacher checkpoint complete.

## Communicating with Teammates
- [ ] Recipe stays visible.
- [ ] Use "behind," "hot," "heard," and "I need help."
- [ ] Reader tracks the next step.
- [ ] Steward starts cleanup early.

## Lab Behavior
- [ ] Read before acting.
- [ ] Organize before cooking.
- [ ] Watch heat and steam.
- [ ] Clean as you go.
- [ ] Ask before changing equipment or heat.

## Cleanup and Reset
- [ ] Dishes routed correctly.
- [ ] Counters cleaned and sanitized.
- [ ] Tools returned.
- [ ] Ingredients stored or returned.
- [ ] Towels handled correctly.
- [ ] Trash and floor checked.
- [ ] Heat off as directed.
- [ ] Station ready for the next group.

## Exit Routine
- [ ] Station inspected.
- [ ] Reflection completed if assigned.
- [ ] Belongings collected.
- [ ] Leave calmly when dismissed.
