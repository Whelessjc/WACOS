# Checklist: Bistro Systems Reinforcement

## Readiness Checklist
- [ ] I know what happened in soft opening.
- [ ] I know my station role.
- [ ] I know my improvement target.
- [ ] I know the prep or service goal for the day.
- [ ] I know how my station affects the guest or next teammate.

## Daily Habits
- [ ] Join huddle.
- [ ] Use call-backs.
- [ ] Keep station clean.
- [ ] Check ticket status.
- [ ] Help reset.
- [ ] Notice dish, towel, floor, and trash needs before the end.
- [ ] Leave my station useful for the next teammate.

## Skill Checks
- [ ] I can explain ticket flow.
- [ ] I can set up my station.
- [ ] I can complete a prep task.
- [ ] I can communicate with expo or teammates.
- [ ] I can debrief honestly.
- [ ] I can describe one hospitality habit.

## Cleaning and Reset Checks
- [ ] Station cleaned.
- [ ] Tools returned.
- [ ] Dishes handled.
- [ ] Towels managed.
- [ ] Floor checked.
- [ ] Service area reset.
- [ ] Storage and labels checked if prep was completed.

## Student Self-Check
- [ ] I improved one service habit.
- [ ] I helped my team.
- [ ] I know what still needs practice.
