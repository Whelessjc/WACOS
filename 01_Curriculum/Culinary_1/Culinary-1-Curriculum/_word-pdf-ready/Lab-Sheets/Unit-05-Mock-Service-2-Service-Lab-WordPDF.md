# Unit 05 Mock Service 2.0 Service Lab

Name: ________________________________    Date: ________________    Class Period: ________

## Purpose
Practice the service systems that need repetition after the soft opening: roles, tickets, communication, recovery,
dish, and reset.

## Skills Practiced
- soft opening debrief
- role clarity
- ticket flow
- communication
- mock service 2.0
- prep
- dish flow
- reset

## Safety Reminders
- Wash hands when directed.
- Walk with purpose and keep paths clear.
- Use "hot," "behind," and "corner" when needed.
- Keep clean and dirty items separated.
- Report spills or unsafe movement right away.

## Setup Checklist
- [ ] Know your station or role.
- [ ] Know the ticket/task path.
- [ ] Set tools, towels, sanitizer, and dish/reset plan.
- [ ] Ask questions before service starts.

## Procedure
1. Complete entry routine, huddle, and handwashing.
2. Review the soft opening debrief focus.
3. Confirm role, station, ticket path, and reset responsibility.
4. Run mock service 2.0 using teacher-provided tickets/tasks.
5. Use call-backs and ask for help early.
6. Pause, name the problem, and recover when needed.
7. Complete service reset and dish flow.
8. Debrief what improved and what still needs practice.

## Teacher Checkpoints
- [ ] Roles are clear before service starts.
- [ ] Tickets move to the correct place.
- [ ] Students use call-backs and useful communication.
- [ ] Recovery steps are calm and specific.
- [ ] Reset is complete before dismissal.

## Cleanup / Reset Checklist
- [ ] Tickets or tasks accounted for.
- [ ] Guest/service area reset.
- [ ] Dishes routed correctly.
- [ ] Counters cleaned and sanitized.
- [ ] Tools returned.
- [ ] Towels handled correctly.
- [ ] Trash and floors checked.
- [ ] Station ready for the next group.

## Reflection Questions
1. What role did you practice?
2. Where did communication help?
3. Where did the system slow down?
4. What cleanup/reset job did you complete?
5. What should we repeat next time?
