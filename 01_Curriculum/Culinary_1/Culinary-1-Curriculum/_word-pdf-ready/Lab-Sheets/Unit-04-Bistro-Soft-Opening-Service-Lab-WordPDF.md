# Unit 04 Bistro Soft Opening Service Lab

Name: ________________________________    Date: ________________    Class Period: ________

## Purpose
Practice a controlled soft opening where students learn the full flow of service from guest arrival to farewell and
reset.

## Skills Practiced
- Life of an Order
- station roles
- guest interaction
- ticket flow
- expo
- food running
- dish flow
- reset

## Safety Reminders
- Wash hands when directed.
- Walk with purpose and keep paths clear.
- Use "hot," "behind," and "corner" when needed.
- Keep clean and dirty items separated.
- Report spills or unsafe movement right away.

## Setup Checklist
- [ ] Know your station or role.
- [ ] Know the ticket/task path.
- [ ] Set tools, towels, sanitizer, and dish/reset plan.
- [ ] Ask questions before service starts.

## Procedure
1. Complete entry routine, huddle, and handwashing.
2. Review station assignments and Life of an Order.
3. Set up station tools, tickets/tasks, towels, and reset plan.
4. Run mock or controlled service as directed.
5. Communicate early and stay in role.
6. Pause for teacher coaching if the system breaks down.
7. Complete dish, guest area, service area, and station reset.
8. Debrief with facts, not blame.

## Teacher Checkpoints
- [ ] Station assignments are clear.
- [ ] Ticket/task flow is visible.
- [ ] Students use guest-appropriate language.
- [ ] Dish/reset is active during service.
- [ ] Debrief identifies one system to improve.

## Cleanup / Reset Checklist
- [ ] Tickets or tasks accounted for.
- [ ] Guest/service area reset.
- [ ] Dishes routed correctly.
- [ ] Counters cleaned and sanitized.
- [ ] Tools returned.
- [ ] Towels handled correctly.
- [ ] Trash and floors checked.
- [ ] Station ready for the next group.

## Reflection Questions
1. What role did you practice?
2. Where did communication help?
3. Where did the system slow down?
4. What cleanup/reset job did you complete?
5. What should we repeat next time?
