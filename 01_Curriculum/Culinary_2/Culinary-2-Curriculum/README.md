# Culinary 2 Curriculum Package

This folder is the organized teaching package for Culinary Arts Management 2.

It does not replace the master planning files one level up. It turns those files into a course structure that can be built out to match the completeness of Culinary 1.

## Current Status

Culinary 2 is currently strong as a framework and pacing system. It still needs the same level of finished classroom materials that Culinary 1 has: daily lessons, lab packets, student handouts, teacher notes, slide outlines, station checks, and export-ready resources.

This package is the build-out location for that work.

## Build Standard

Each unit should eventually include:

- teacher overview
- student-facing unit packet
- weekly agenda set
- lab plans
- practical checks
- line check prompts
- assessment language
- Bistro connection
- CKC evidence connection when appropriate
- recipe vehicles
- reflection prompts grounded in performance

Keep the course centered on fundamentals under pressure. Recipes are vehicles. Bistro is the proving ground.
