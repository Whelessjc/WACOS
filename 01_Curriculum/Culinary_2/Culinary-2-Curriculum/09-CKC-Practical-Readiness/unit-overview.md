# Unit 9: CKC Practical Readiness

Approximate timing: Embedded, with focused pushes

## Purpose

Certified Kitchen Cook readiness should be built through actual kitchen work, not isolated test-prep days. Students gather evidence through practical skills, line checks, service habits, sanitation, and composed execution.

## Primary Sources

- `../../05_CKC_Alignment.md`
- `../../../03_Assessment/Culinary_2_Line_Check_Cards.md`
- `../../../03_Assessment/Culinary_2_Assessment_and_Rubrics.md`

## Learning Priorities

- Connect daily labs to CKC skill evidence.
- Answer short oral checks clearly.
- Demonstrate safe and useful kitchen behavior.
- Complete practical tasks under reasonable pressure.
- Use feedback to improve performance.

## Suggested Applications

- line checks
- station checks
- thermometer checks
- knife and prep practicals
- sanitation observations
- service evidence
- short scenario responses

## Evidence

Students should show practical readiness through repeated performance, not one polished event.

## Build-Out Notes

Create CKC evidence trackers, practical check calendars, line check sets by unit, and student readiness conference forms.
