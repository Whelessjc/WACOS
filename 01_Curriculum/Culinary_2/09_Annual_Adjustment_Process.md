# Annual Adjustment Process

This curriculum is meant to be maintained year to year.

Do not rebuild it from scratch unless the course identity changes.

## Before the Year Starts

Update:

- school calendar
- testing windows
- early release dates
- known field trips
- guest speakers
- pep rallies or major school events
- Hospitality Scholars Tuesdays
- Bistro start date
- CKC target window

Then choose:

- anchor recipes for each module
- Bistro opening menu
- likely Semester 2 baking products
- capstone format options

## Quarter Review Routine

At the end of each quarter, answer:

- Which routines held?
- Which routines drifted?
- Which skill needed more reps?
- Which recipe helped?
- Which recipe wasted time?
- What did Bistro expose?
- What should move earlier next year?
- What can be cut?

Keep notes short.

## What to Protect Every Year

Protect:

- stable beginning routine
- stable ending routine
- Thursday Bistro prep
- Friday Bistro service
- knife control
- recipe reading
- clean-as-you-go
- tasting and seasoning
- communication
- station reset
- composure under pressure

## What Can Change

Change:

- recipes
- Bistro menu
- capstone format
- exact week placement
- number of reps per skill
- assessment check dates
- Semester 2 module order

## CKC Review

After the CKC window, note:

- Was Week 9 realistic?
- Which CKC skills were strong?
- Which skills needed more kitchen evidence?
- Did students understand the credential as part of the kitchen work?
- What line checks should be added earlier?

## Bistro Review

After each month of Bistro, note:

- strongest station
- weakest station
- recurring prep issue
- recurring communication issue
- product quality issue
- sanitation issue
- one change for next month

## End-of-Year Reset

At the end of the year:

1. Save the working version.
2. Mark recipes as keep, revise, or replace.
3. Update line check cards.
4. Update Bistro systems.
5. Update Semester 1 pacing notes.
6. Leave clear notes for next year's start.

Do not over-polish the documents. They should stay usable in a real culinary lab.
