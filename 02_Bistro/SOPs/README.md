# Bistro SOPs

Current as of July 2026. These documents consolidate real training materials and side-work checklists into the current WACOS structure.

## Current Documents

| Document | Purpose |
| --- | --- |
| `Life_of_an_Order.md` | Full guest-to-reset Bistro flow |
| `Service_Standards.md` | Greeting, serving, clearing, timing, shorthand, and HEARD recovery |
| `Dining_Team_SOP.md` | Host, server, and expo opening/service/closing duties |
| `Kitchen_Team_SOP.md` | Cook station setup, menu production, and closing duties |
| `Support_Team_SOP.md` | Dish room and stewarding procedures |
| `Barista_Team_SOP.md` | Barista setup, service, close, and core drink builds |

## Team Names

Use Dining, Kitchen, Support, and Barista consistently. Older Management Staff and Service Staff labels are retired.

## Confirmation Boundary

These SOPs document current training practice. Payment types, reservation-versus-queue workflow, refunds/comps, cash handling, and till reconciliation remain instructor/admin confirmation items before publication as final policy.
