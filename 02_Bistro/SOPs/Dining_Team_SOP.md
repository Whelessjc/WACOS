# Dining Team SOP (Server, Host, Expo)

Current as of 2026-07. Source: real Server Side Work Checklist in current use. See `Service_Standards.md` for the greet/serve/clear/complaint standards this team applies during service.

## Opening Duties

### General Setup and Cleanliness

- Host stand, server station, and expo stand are clean, sanitized, and organized.
- Foyer and dining room are clean and clutter-free.
- All tables, chairs, and booths are balanced, cleaned, and sanitized.
- Dining room floors are swept.
- Centerpieces have flowers and are arranged.
- Background music is playing.

### Reservations and Menu

- Confirm any special requests with chef and manager.
- Print menus; confirm they're clean and in order.
- Menu is updated and posted (felt letter board, printed, QR code, or Promethean board).
- Reservations sheet is available and filled out.
- Memorize specials and be ready to communicate them to guests.

### Server and Host Station Prep

- Bistro neon light on.
- Square POS system turned on and updated; confirm a stable internet connection.
- Printer paper stocked for receipts.
- Check menu items in Square to confirm accuracy.
- Order tablets, tickets, and pens ready.
- Clean server apron (solid black).

### Service Essentials

- Utensils and glasses polished; utensils rolled as needed.
- Clean napkins available.
- Straws and souffle cups ("silver bullets") stocked.
- Take-out containers and packaging stocked.
- Drink and food trays/stands available.

### Expo Station Setup

- Expo station clean and organized (pens, sharpies, highlighters).
- Foyer (kitchen door) clean and organized.
- Copy of the menu written and displayed.
- Reservations sheet on station (Chromebook).
- Kitchen Manager's log on station.
- Phone etiquette practiced; phone directory posted near phone.
- Trash can at station.
- Vinegar water and buffing cloth available.
- Hot plates ready for service.
- Take-out supplies stocked (1 oz and 2 oz souffle cups with lids, and other packaging as needed).

## During Service

- Run the real steps of service: host greets and confirms party size/reservation, seats the table, hands off to the server by name. Server introduces themselves, mentions the special, offers a drink.
- Apply the 30-Second Greet, table setting standard, and Left-Lower-Leave serving from `Service_Standards.md`.
- Use order shorthand accurately; write out anything unusual in full.
- Track service time benchmarks; communicate early if a table is falling behind.
- Flag allergies and dietary restrictions to the kitchen immediately - never guess.
- Use HEARD if a guest raises a complaint directly.

## Closing Duties

- Clear all tables from the right; don't stack dishes at the table.
- Reset dining room for the next service - tables, chairs, floors, host stand.
- Square POS closed out per manager instruction.
- Return reservations sheet, Kitchen Manager's log, and any borrowed equipment to their stations.
- Restock take-out supplies and receipt paper for next service.
- Report any equipment issues (POS, printer, neon light) to the instructor.
