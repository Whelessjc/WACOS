# Program Administration

## Purpose
Document administrative systems that support program stability, compliance, funding, partnerships, scheduling, and growth.

## Program Assets
- West Ashley Culinary identity.
- Kitchen + Classroom public frame.
- Bistro as student-led restaurant experience.
- Culinary 1, Culinary 2, and Hospitality Work-Based Learning pathway.
- Credential pathway including Food Handler and CKC planning.
- Public website presence with Home, About, Recipes, Courses, Gallery, Press, Contact Info.

## Administrative Areas
| Area | Documentation Needed |
|---|---|
| Budget | Funding sources, Bistro revenue use, spending rules, recurring costs |
| Ordering | Vendors, order timeline, receiving, inventory, substitutions |
| Equipment | Inventory, maintenance, replacement priorities |
| Certifications | Food Handler, CKC, any ServSafe Manager decision, testing calendar |
| Partnerships | Explore Charleston, Trident Tech/apprenticeship, local restaurants, guest speakers |
| Competitions | Eligibility, coaching calendar, travel, fundraising |
| Work-Based Learning | On-campus/off-campus options, hours, documentation, expectations |
| Public presence | Website accuracy, press page, recipe archive, contact information |

## Hospitality Scholars
The source history identifies High School Hospitality Scholars with Explore Charleston on the last Tuesday of each month. Curriculum plans should protect flexibility around those dates.
