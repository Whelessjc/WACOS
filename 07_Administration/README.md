# 07_Administration

This folder contains administrative and program-management documents.

## Belongs Here

- Program administration.
- Teacher assistant onboarding.
- Future development.
- Partner documentation.
- Budget and vendor workflows.
- Certification calendars.
- Work-Based Learning administration.
- Public presence planning.

## Does Not Belong Here

- Daily lesson plans.
- Recipes.
- Raw archive transcripts.
- Equipment SOPs.
- Student-facing handbooks unless administration is the main purpose.

## Relationship to the Core Manual

Administration documents should support the program identity, current credential pathway, Bistro model, and documentation governance described in the Core Manual.

## Caution

Budget, vendor, partner, certification, and WBL procedures may require school or district approval.
