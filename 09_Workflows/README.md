# Instructor Workflows

## Purpose

This folder documents how the instructor operates West Ashley Culinary day to day and week to week. It connects planning, teaching, Bistro operations, assessment, AI-supported work, administration, and reflection in one instructor-facing workflow layer.

This folder is not curriculum, policy, a student handbook, or a replacement for current SOPs. It explains how the instructor uses the existing WACOS system to do the work.

## Operating Boundaries

- Current instructor-confirmed decisions and the [Core Manual](../00_Core/West_Ashley_Culinary_Core_Manual.md) govern this folder.
- [Monday through Wednesday instruction](../00_Core/WACOS_Playbook.md#monday-wednesday-instructional-rhythm) is protected whenever the calendar allows.
- Thursday is Bistro Prep.
- Friday is Bistro Service.
- Student-facing materials require instructor review.
- District systems remain external unless a current, district-approved connection is available.
- Permanent WACOS changes follow the [WACOS Update Workflow](../00_Core/WACOS_Update_Workflow.md).

## Maintenance

Update these workflows when actual instructor practice changes. Do not use a workflow edit to create grading, curriculum, Bistro, safety, district, or cash-handling policy. Link to the governing document and flag unresolved details for instructor confirmation.
